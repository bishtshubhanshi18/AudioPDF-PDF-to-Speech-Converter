# AudioPDF-PDF-to-Speech-Converter
Description
AudioPDF is a Python desktop application that converts PDF documents into spoken audio. It allows users to listen to books, reports, or any PDF content, providing a hands-free reading experience. The application uses PyPDF2 for text extraction and pyttsx3 for offline text-to-speech conversion, with a Tkinter GUI for easy interaction.
